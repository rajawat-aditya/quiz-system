# 📝 Quiz System

A Laravel-based Quiz Management System where users can attempt quizzes and admins can manage categories, quizzes, and MCQs.

---

## 🚀 Features

### 👤 User
- Register & login  
- Browse quizzes by category  
- Attempt MCQ-based quizzes  
- Instant result & score  
- View quiz history  

### 🔐 Admin
- Admin dashboard  
- Manage categories  
- Create & manage quizzes  
- Add/Edit/Delete MCQs  
- View users and performance  

---

## 🛠️ Tech Stack
- Laravel (PHP Framework)  
- PHP  
- MySQL  
- Blade  
- Tailwind CSS
